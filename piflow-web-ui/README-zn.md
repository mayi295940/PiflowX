# PiFlow大数据流水线管理系统 Web 端
支持pc端自适应。

[English](./README.md) | 简体中文

### 项目初始化
```
npm install
```

### 启动开发环境
```
npm run dev
```

### 项目打包上线
```
npm run build
```

### 项目优化可视化分析
```
npm run analyzer
```

### Lints和fixes文件
```
npm run lint
```
